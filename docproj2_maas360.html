MaaS360 On-premise Cloud Extender Installation Manual

Context

MaaS360 on-premise cloud extender was conceptualized and launched to support enterprises who were looking for options for an on-premises version of the mobile security solution.

Opportunity

Since I had experience working with the SaaS version of the MaaS360 cloud extender tool, I was brought on the team by my manager to develop the installation manual for the on-premise version. I was the lone writer on the team.

Contribution

Using my past tool knowledge, I asked questions to explore the behavior of the tool for different scenarios. Since the setup was done using virtual machines and had limited accessibility, I studied the Jira tickets and Confluence pages to learn more about the different features and functionalities. When needed, the developers demonstrated the behavior of the tool, and assisted in collecting screenshots for the installation manual. I worked with product managers to create architecture diagrams, and the support team to add troubleshooting details for common scenarios for a good customer experience.

Impact

  • Created an installation manual from scratch with cross-collaboration from different teams
  • Reviewed customer questions about the MaaS360 On-premises Cloud Extender application to draft an FAQ document after the manual was completed.
  • Received recognition from the management for single-handedly managing the efforts along with other deliverables.
