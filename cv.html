Open Source Contributions

CircuitVerse, Google Season of Docs (GSoD) 2020

Context

I participated in Google Season of Docs (GSoD) in June 2020 to improve my technical writing skills and learn more about contributing to open source projects.

Project Scope

In my GSoD proposal for CircuitVerse (CV), I reviewed and audited the available documentation, and proposed to:

  • Document new features and update documentation for existing features.
  • Improve individual pages and sections for clarity and concision.
  • Add relevant images, gifs, and multimedia to enrich and engage readers.
  • Review the existing content structure for improved consistency, readability, and cohesiveness.
  • Create video tutorials for improving the onboarding experience.

Key Learnings and Deliverables

When GSoD concluded in December 2020, I had produced video tutorials, and rewritten the complete user documentation for the CV simulator.

Following are some of my accomplishments over the four months:

  • Produced one explainer video and six instructional videos, with voice-over and organization branding, for improving the onboarding experience of CircuitVerse(CV) users. These videos can be reused by the educators (primary audience) while planning their classroom assignments.
  • Revised the entire user documentation (approx. 130 pages) for the CV simulator interface by adding content for new features and functionalities, incorporating figures, tables, and GIFs as visual aids for concept illustration.
  • Studied the queries received across CV Slack channel and CV forums to understand user pain points, and added different support resources to take the burden off the CV mentors.
  • Conducted a gap analysis and categorized existing content into meaningful content types.
  • Improved the content hierarchy and flow by adding chapters and subchapters for making the content more searchable.
  • Generated markdown files for the user documentation before uploading it into GitHub.
